❌ Issue: Low-Contrast Link Color

All links on this page use color: #aaa on a white background. That gives a contrast ratio of approximately 2.3:1, far below the WCAG AA requirement of 4.5:1 for normal-sized text. Users with low vision cannot reliably read these links. Fix: use a darker color such as #767676 or #0060df.

❌ Issue: Missing Alt Attribute on Image

The news thumbnail above has no alt attribute at all. A missing alt is different from an empty one — assistive technologies will often read out the image file path or URL as a substitute, which is meaningless. Fix: add a brief description, e.g. alt="Abstract blue chart graphic", or alt="" if the image is decorative.

❌ Issue: Navigation Not in a <nav> Landmark

The top navigation links are wrapped in a plain <div> rather than a <nav> element. Screen reader users use landmark navigation to jump directly to the main navigation area of a page. Without a <nav> element, that shortcut doesn't work. Fix: wrap the links in <nav aria-label="Main">.
